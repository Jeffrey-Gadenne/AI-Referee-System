# AI Referee System

**AI-powered real-time detection and referee system** built for automated rule enforcement and behavioral analysis.

## 🎯 Project Overview

Production-grade computer vision and sensor fusion system using Raspberry Pi 5. Demonstrates core skills directly applicable to **anti-cheat systems**, game security, and behavioral threat detection.

## ✨ Key Features

- Real-time multi-object detection and tracking
- Pose estimation and movement validation
- BLE sensor integration for precise tracking
- Behavioral anomaly detection engine
- Live dashboard with event logging
- Automated alert and rule enforcement system

## 🛠️ Tech Stack

- **Python** 3.11+
- **OpenCV** + **MediaPipe** for computer vision
- **YOLO** models (optional)
- Raspberry Pi 5 + Camera Module
- BLE / IoT sensors
- Docker for deployment
- Streamlit or Flask for dashboard

## 🚀 Quick Start

```bash
git clone https://github.com/Jeffrey-Gadenne/AI-Referee-System.git
cd AI-Referee-System
pip install -r requirements.txt
python main.py
```
